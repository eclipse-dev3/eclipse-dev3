💫 About Me
Frontend Developer passionate about building scalable, responsive web applications with clean code and exceptional user experiences.
I work with React.js, Redux Toolkit, Context API, Tailwind CSS, and Vanilla JavaScript to craft modern Single Page Applications. I have built OwnDocs — a full-featured Note & Task Manager with folder organization, soft delete, debounced search, and offline-first LocalStorage persistence — and MusicX — a Spotify-inspired music player with dynamic playlist rendering and real-time playback controls.
Strong understanding of JavaScript concepts including Closures, Async/Await, Event Loop, DOM Manipulation, and Debouncing. Proficient with Git, GitHub, VS Code, Vercel, and Figma.
Currently pursuing my goal of becoming a Full Stack Developer — always learning, always building.

"Consistent effort and clean code build solutions that make a real difference."
